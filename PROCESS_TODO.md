# Process TODO

1. Open TODO.md
2. Find the next task that is not completed [ ] or [-]
3. Mark ONE Task as [-] (in progress)
4. Complete Task
5. Mark as Complete [x]

## NOTE

Run git diff [FILE NAME] to see changes that have already been made.

Run pytest to see if codes has errors running

If a Task is difficult first add your Intetentions to TODO.md and then start fixing
